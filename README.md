# Serra Vano Milano

L'estetica dell'ostacolo

## About

Serra Vano is a conceptual art project exploring luxury kitchenware that transcends traditional function. Each piece challenges conventional design by prioritizing form over utility.

## Features

- Modern, responsive design
- Product gallery with filtering
- Individual product detail pages
- Philosophy and about sections

## Structure

- `index.html` - Homepage
- `products.html` - Product gallery
- `product-detail.html` - Product detail page
- `about.html` - Philosophy page
- `styles.css` - All styling
- `app.js` - Interactive functionality
- `products.json` - Product data
- `images/` - Product images and videos

## Deployment

This is a static website designed for GitHub Pages. Simply push to your repository and enable GitHub Pages in settings.

## Local Development

Open `index.html` in a web browser to view locally. For best results, use a local development server.
